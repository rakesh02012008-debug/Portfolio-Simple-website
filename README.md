

This portfolio website is created using **HTML** and **basic CSS**, focusing on simplicity and clarity. The structure of the website is built with HTML, which defines the content and layout of the pages, including sections like the Home page and Contact page. CSS is used to style the website, adding colors, spacing, fonts, and basic layout design to improve the visual appearance.

The website consists of multiple pages connected through simple navigation links, allowing users to move easily between sections. The design follows a minimal approach, making it easy to understand and accessible for beginners.

This project demonstrates fundamental web development concepts such as page structure, styling, and multi-page navigation, making it a great starting point for learning front-end development.
